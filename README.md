##  História do Jogo


O jogador assume o papel de Steve, herói em um mundo de blocos que precisa sobreviver às caos da dimensão THE END. Bafo do dragao do End se aproximam continuamente pela neblina, e você deve esquivar com rapidez e precisão para sobreviver o máximo de tempo possível. 

Cada desvio bem-sucedido aprofunda o desafio - Os bafos do dragão do End ganham velocidade, testando seus reflexos e estratégia. O objetivo é atingir a maior pontuação possível enquanto enfrenta ondas cada vez mais intensas de perigo, competindo pela posição no hall da fama global do jogo.

Um jogo perfeito para quem ama ação rápida, desafios progressivos e o icônico estilo visual de blocos do Minecraft.

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias e bibliotecas:

- **Python 3.12.13** - Linguagem de programação principal
- **Pygame** - Engine gráfica para renderização de sprites, animações, colisões e eventos de entrada
- **JSON** - Formato para persistência de dados de pontuação e rankings
- **cx_Freeze** - Ferramenta para compilação e distribuição do jogo como executável independente (.exe)

### Características Técnicas:

- **Detecção de Colisão**: Sistema baseado em pixel-perfeito para detecção precisa entre o jogador e obstáculos
- **Dificuldade Dinâmica**: Progressão automática de dificuldade conforme o jogador avança
- **Sistema de Audio**: Efeitos sonoros e trilha sonora para feedback auditivo
- **Persistência de Dados**: Banco de dados local armazenando histórico de pontuações e rankings
- **Interface Gráfica**: Menus intuitivos e HUD em tempo real
